# Animali API

## REST API Specification

| Ednpoint     | HTTP Method | Description      |
| ------------ | ----------- | ---------------- |
| /animals     | GET         | Get all animals  |
| /animals/:id | GET         | Get animal by id |

## Getting Started

To install dependencies:

```sh
bun install
```

To run:

```sh
bun run dev
```

open http://localhost:3000
